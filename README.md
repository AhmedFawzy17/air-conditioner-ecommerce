# Air Conditioner E-commerce

مشروع متجر إلكتروني متكامل لبيع أجهزة التكييف، تم تصميمه وتطويره باستخدام **React + TypeScript** و **Vite**.

---

## 🚀 المميزات والتقنيات

- **Frontend:** تم بناء الواجهة باستخدام React مع TypeScript لكتابة كود منظم وقابل للتوسع.
- **Build Tool:** استخدام Vite لتسريع عملية التطوير ورفع الأداء.
- **Responsive Design:** تصميم متجاوب يعمل على جميع الأجهزة (موبايل - تابلت - ديسكتوب).
- **API Integration:** ربط المشروع بـ Backend لإدارة المنتجات والطلبات.

---

## ⚙️ كيفية تشغيل المشروع

لتشغيل المشروع محليًا اتبع الخطوات التالية:

1. تثبيت dependencies:

